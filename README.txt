STEINBECK RESTORED FULL PACKAGE

Contents
--------
1. steinbeck_lemons_full_research_notebook.ipynb
   Full Jupyter notebook with:
   - theoretical markdown cells
   - executable code cells
   - interactive sliders via ipywidgets.interact_manual
   - figures saved into figures/
   - tables saved into tables/
   - comparative scenarios
   - institutional sensitivity experiment

2. paper_aer_style.tex
   Full AER-style paper draft in English.

3. references.bib
   Bibliography file.

4. requirements.txt
   Python dependencies.

How to use
----------
1. Install the requirements.
2. Open the notebook in Jupyter.
3. Run all cells from top to bottom.
4. This will create figure and table files in the package folders.
5. Compile paper_aer_style.tex after the notebook has generated the outputs.

Notes
-----
- The notebook uses one period = one week.
- To avoid repeated heavy recomputation, the interactive block uses a Run button.
- All content is in English.
